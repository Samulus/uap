About uap:

uap is web based media player that allows you to stream music from your computer to any computer running a modern web browser. It's a work in progress at the moment but we are excited to release it this December.

Usage:

Simply launch the main uap.py file and connect to http://127.0.0.1:8080 in your web browser, the first time you launch the program you will be greeted with a configuration page. Otherwise you're all set and can start listening to your music anywhere.

Technologies used:

uap makes use of several excellent open source libraries to make music streaming possible including:

Backend

  • bottle.py as a portable webserver / framework
  • SQLite3 for serializing music library information offline / quick lookups
  • pytaglib / Taglib for collecting metadata from music files

Frontend
